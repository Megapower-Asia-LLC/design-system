# Colors

群兆 design token 色票。生成 UI 時一律用 `var(--…)`，不要寫死色碼。

- 品牌：`--color-primary` #F06000（官方 logo 橘）、`--color-primary-hover` #D45200、`--color-primary-soft-bg` #FFF7ED
- 文字：`--color-text` #1E293B、`--color-text-muted` #64748B
- 表面：`--color-bg` #FFFFFF、`--color-bg-soft` #F8FAFC、`--color-border` #E2E8F0
