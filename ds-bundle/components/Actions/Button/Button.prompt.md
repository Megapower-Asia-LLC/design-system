# Button

群兆按鈕。基底 `.btn` + 樣式修飾 + 尺寸修飾。

- 樣式：`.btn--primary`（實心品牌橘，主要行動）｜`.btn--secondary`（白底灰框，次要）｜`.btn--outline`（橘框，第三層）
- 尺寸：`.btn--sm` / `.btn--md` / `.btn--lg`
- 在深底區塊 `.section--dark` 內，`.btn--secondary` 自動轉淺色描邊。

```html
<a class="btn btn--primary btn--lg">開始分析</a>
<a class="btn btn--outline btn--md">了解更多</a>
```
