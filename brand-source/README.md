# brand-source — 品牌資產原始檔備援

logo 向量原檔（Adobe Illustrator）備援副本。工作正本在 OneDrive（群兆資訊）`行銷相關/Logo/`；
此處為 git 版本化備援——原檔遺失即無法重製 `ds-bundle/logo/` 的輸出 PNG，故入 repo 保底。

- `logofinal 01*`：主標誌定稿家族（含 RGB 版與文字轉外框 out 版——印刷/交付用外框版）
- `logofinal 02*`：完整標誌（M + 公司名 + 標語）定稿家族

本目錄**不進 npm 套件**（package.json files 白名單外），僅存於 repo。

現況：六個 `.ai` 原檔已入庫（2026-07-09，自 OneDrive 複製、檔頭 `%PDF-1.4` 驗證完整）。
更新原檔時：在 OneDrive `行銷相關/Logo/` 改好 → 複製覆蓋此目錄同名檔 → commit（無自動同步，人工帶）。
輸出規範（尺寸/留白/誤用）見 `ds-bundle/BRAND-ASSETS.md` 與 `BRAND-GUIDELINES.md`。
